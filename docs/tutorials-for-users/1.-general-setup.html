1. General Setup

1. Install Database Designer

You can find Database Designer on

  • Steam or
  • Itch.io

2. Starting up the program

  1. Start by launching Database Designer
    1. For Steam, you can simply head to your library and click "Start".
    2. For Itch.io, go to the location you extracted the .zip file to and click "Database Designer.exe"
  2. Leave the opening CMD to startup; this is completely part of Database Designer and handles a few things at the backend

At this point, you will be at the starting screen!

ðŸ’œ Hotkeys Exist In Database Designer!

Database Designer has two shortcuts you can use from the start!

Alt + M will (m)ute the sounds
Alt + L will toggle the (l)ower bar's visibility


### 3. Creating an account
  1. To create an account with Database Designer, start by clicking the login button and create user.

  1. Now, set a username and password. The password must have an uppercase and lowercase letter, a symbol and be 12+ characters long. It's important to click "Continue" when done and not just switch to another tab.

ðŸ’œ Switching Tabs Resets Login Data!

It may not look like it, but every time you switch back to the Basics Tab, the Username and Password information is reset!
You will have to reenter the information and click continue.

"But Why?" Mainly to help make sure your credentials are secured!


  1. Select an avatar PFP and click continue.

  2. Validate the information entered; if it looks good, click Finalize.

  3. Take the code from the next page and save it somewhere secure. In future updates, this code will allow you to reset your password. Do not share this code with ANYONE.
