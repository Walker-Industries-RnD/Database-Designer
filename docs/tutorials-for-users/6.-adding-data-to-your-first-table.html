6. Adding Data To Your First Table

Navigating and Editing Tables in OpenSilver

Traversing the Table Editor

To work with a specific table:

  1. Open your project in the OpenSilver Simulator or the drag-and-drop XAML Designer.
  2. Navigate through the visual tree or editor interface to locate the desired table item (e.g., a DataGrid instance).
  3. Click the table item to select it.

Once selected, you'll gain access to advanced editing options, including adding new data structures. You can view values to the middle row and the detailed information for each to the right. You also have the option to delete and edit rows as you please.

Adding New Data

When a table is selected, you have three primary options for extending its structure:

1. Adding a New Column

  • This creates a standard column for holding data fields or attributes.
  • Ideal for adding simple properties like text, numbers, dates, or custom templates.
  • After adding, configure the column's header, binding, width, and editing behavior in the properties panel.

2. Adding an Index Option

  • Enables indexing on the column for improved performance, uniqueness enforcement, or sorted display.
  • Useful for primary keys and search optimization
  • Configure index type (e.g., unique, non-unique) and apply it to one or more columns.

3. Creating Reference Columns

  • Allows defining relationships between tables (similar to foreign keys).
  • Supports data integrity by linking to primary keys in related tables.
  • Great for relational data models, such as linking a "Orders" table to a "Customers" table.

Deleting Your Table

To delete your table, you can click the "Delete Table" button twice within 10 seconds.
