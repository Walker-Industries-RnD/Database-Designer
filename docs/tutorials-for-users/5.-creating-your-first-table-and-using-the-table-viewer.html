5. Creating Your First Table And Using The Table Viewer

To begin, start by clicking "Table Editor". You will see a UI similar to the file editor appear.

Sometimes, it might be hard to see what's happening because of what's going on in the background!
To change this, slide the slider at the top right to change the transparency of the background!
You can change transparency for windows separately, allowing you to prioritize some and ignore others if need be!

Click the + button to open the table creation UI.
You have the option of creating a completely new table and schema, or using a template.

There must not be a table with the same name at the same schema.
Templates with names will use their own table name unless it's taken at the area; at that point the table name you enter will be added in!

Click Default to clear any selected template and return to manual table creation.
